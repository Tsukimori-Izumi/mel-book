## AC01: MLアーキテクチャを作成する

### TA0101: 前処理・処理・後処理の仕様化

```mermaid
flowchart LR
  A[整数入力 1〜100] --> B[ルールエンジン]
  B --> C[結果文字列]
  C --> D[出力\nコンソール+ファイル]
```

| 段階 | 仕様 |
|---|---|
| 前処理 | なし（整数をそのまま使用） |
| 処理 | if/elif条件分岐（15→FizzBuzz、3→Fizz、5→Buzz、他→数値） |
| 後処理 | 結果文字列のフォーマット "[数値]: [結果]" |

### TA0102: コンポーネントの検討

| コンポーネント | 責務 | 技術 |
|---|---|---|
| **RuleEngine** | FizzBuzzルール判定 | Python if/elif |
| **OutputFormatter** | コンソール＋ファイル出力 | Python print/open |

### TA0103: ハイパーパラメータ

ルールベースのためハイパーパラメータは**なし**。  
設定パラメータ:

| パラメータ | 値 |
|---|---|
| 入力範囲 | 1〜100 |
| 出力ファイル | `fizzbuzz_result.txt` |

### TA0104: アーキテクチャエレメントの仕様化

```
入力: 整数 n (1〜100)
  ↓
判定ロジック:
  if n % 15 == 0 → "FizzBuzz"
  elif n % 3 == 0 → "Fizz"
  elif n % 5 == 0 → "Buzz"
  else → str(n)
  ↓
出力: コンソール + fizzbuzz_result.txt
```
