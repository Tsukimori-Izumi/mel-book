# MLE1 アクティビティ詳細リファレンス

MLE1（機械学習要求分析）プロセスのアクティビティ詳細です。

---

## MLE1.AC01: 機械学習要求の取得と分解

**対応BP**: MLE1.BP1（仕様化）  
**目的**: 機械学習要件を上位要求から取得し、実装可能な粒度に分解する

### 配下タスク

| タスクID | タスク内容 |
|---|---|
| MLE1.TA0101 | 機械学習要件を取得する |
| MLE1.TA0102 | 機械学習要件を分解する |
| MLE1.TA0103 | これまでMLE1で検討した機械学習要件を表にまとめる |

### 入力成果物
- 17-00 要求（上位要求仕様）

### 出力成果物
- 機械学習要件一覧表
- 分解された機械学習要件

---

## MLE1.AC02: 機械学習要求の実現、検証可能性の検証と統合

**対応BP**: MLE1.BP2（運用環境）、MLE1.BP3（優先順位）  
**目的**: 機械学習要件の実現可能性と検証可能性を評価し、優先順位をつけて統合する

### 配下タスク

| タスクID | タスク内容 |
|---|---|
| MLE1.TA0201 | 機械学習要件の運用環境への影響を考慮する |
| MLE1.TA0202 | 機械学習要件の実現方法の可能性を考慮する |
| MLE1.TA0203 | 機械学習要件の検証方法の可能性を考慮する |
| MLE1.TA0204 | 機械学習要件を合成する |
| MLE1.TA0205 | 機械学習要件の優先順位付けを記述する |

### 入力成果物
- MLE1.AC01 の出力（機械学習要件一覧）
- 17-54 要求の属性
- 15-51 分析結果

### 出力成果物
- 実現可能性分析結果
- 検証方法の定義
- 優先順位付き機械学習要件
