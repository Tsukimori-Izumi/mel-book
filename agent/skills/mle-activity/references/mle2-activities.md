# MLE2 アクティビティ詳細リファレンス

## MLE2.AC01: MLアーキテクチャを作成する

**対応BP**: MLE2.BP1（MLアーキテクチャを作成する）  
**目的**: MLモデルの詳細、前処理、後処理を作成し、ハイパーパラメータを定義し、MLアーキテクチャエレメントを仕様化する

### 配下タスク

| タスクID | タスク内容 |
|---|---|
| MLE2.TA0101 | MLモデルの前処理、処理、後処理について仕様化する |
| MLE2.TA0102 | MLアーキテクチャコンポーネントを検討する |
| MLE2.TA0103 | MLモデル、トレーニング、テストに必要となるハイパーパラメータについて仕様化する |
| MLE2.TA0104 | MLアーキテクチャエレメントについて仕様化する |

### 入力成果物
- 17-00 要求、17-54 要求の属性、15-51 分析結果（MLE1出力）

### 出力成果物
- 04-51 MLアーキテクチャ
- 01-54 ハイパーパラメータ
