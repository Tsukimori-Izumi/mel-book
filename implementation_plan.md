
## 各成果物の設計

### アクティビティスキル (`mle-activity`)

**目的**: プロセスのアクティビティを順次実行するガイド  
**トリガー**: 「MLE1のアクティビティを実行」「アクティビティACxx を実行」等  
**内容**:
- 各アクティビティの目的・入出力・手順
- A-SPICE基本プラクティスとの対応
- references/ にプロセスごとの詳細を配置

### タスクスキル (`mle-task`)

**目的**: アクティビティ内の個別タスクを実行するガイド  
**トリガー**: 「タスクTAxxxx を実行」「MLE1.TA0101を実行」等  
**内容**:
- 各タスクの具体的な手順・入出力・成果物
- references/ にプロセスごとのタスク詳細を配置

### プロセスワークフロー (`mle1-process.md`, `mle2-process.md`)

**目的**: プロセス全体をステップバイステップで実行する手順書  
**トリガー**: `/mle1-process`, `/mle2-process`  
**内容**:
- プロセスの全アクティビティを順番に実行する手順
- 各ステップでアクティビティスキルを呼び出す指示
- 入出力成果物の受け渡し
