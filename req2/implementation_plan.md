# SUP11 MLデータ管理 GUI ソフトウェア — 実装計画

## 要求

```
req2: MLE-book(draft).md の SUP11 のすべてのアクティビティ、タスク活動を支援するソフトウェアの開発。
人間がタスクを代替えできるように、GUIを提供してください。
```

## SUP11 構造（6 Activities, 23 Tasks）

| AC | アクティビティ | タスク数 |
|---|---|---|
| AC01 | MLデータ管理システムの確立 | 7 |
| AC02 | MLデータの品質アプローチの作成 | 7 |
| AC03 | MLデータの収集 | 3 |
| AC04 | MLデータの処理 | 3 |
| AC05 | MLデータの品質保証 | 2 |
| AC06 | 処理済MLデータの伝達 | 1 |

## 技術選定

- **HTML + CSS + JavaScript**（単一ファイル、外部依存なし）
- **LocalStorage** でデータ永続化（サーバー不要）
- タブUI で各アクティビティを切り替え
- 各タスクに入力フォーム＋完了チェック＋メモ欄

## 提案するファイル構成

```
req2/
├── req2.md              ← 要求（既存）
├── MLE1-output.md       ← MLE1要求分析
├── MLE2-output.md       ← MLE2アーキテクチャ
└── sup11_manager.html   ← GUIアプリ（単一ファイル）
```

## GUI設計

- **ヘッダー**: タイトル + 全体進捗バー
- **タブナビゲーション**: AC01〜AC06 の6タブ
- **各タブ内**: タスクカード（チェック + メモ + 日時記録）
- **フッター**: エクスポート（JSON）+ リセットボタン
